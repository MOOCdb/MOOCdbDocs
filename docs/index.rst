#################################
MOOCDB Research Guide
#################################

This documentation is for the MOOCdb project. The documentation provides detailed instructions for users to be able to use different 
software and analytic engines developed as part of the MOOCdb project. Our goal is to provide this as a open source repository to 
enable prolific development of MOOC data analytics to improve teaching and learning outcomes. We also refer to this open source 
software repository as MOOC Data Science Commons.
